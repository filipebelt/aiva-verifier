## About the Project

AIVA (Artificial Intelligence Verification Assistant) is a Python-based system that analyzes images and videos to determine whether they were generated by AI models or are authentic.

The system combines computer vision techniques, synthetic pattern analysis, noise detection, and preprocessing pipelines to deliver reliable and interpretable classification results.

---

## Features

- Detection of AI-generated images  
- Video authenticity analysis  
- Probability-based classification  
- Streamlit web interface  
- Dockerized execution  
- Scalable deployment on Google Cloud Run

## Run Locally

```bash
git clone https://github.com/filipebelt/aiva-verifier
cd aiva-verifier
pip install -r requirements.txt
streamlit run main.py
```

---

## Docker

Build and run the container:

```bash
docker build -t aiva-verifier .
docker run -p 8080:8080 aiva-verifier
```

---

## Google Cloud Deployment

Deployment pipeline overview:

1. Build using **Cloud Build**  
2. Store the image in **Artifact Registry**  
3. Deploy to **Cloud Run**

Example deployment command:

```bash
gcloud run deploy aiva-verifier \
  --image=gcr.io/PROJECT_ID/aiva \
  --region=us-central1 \
  --platform=managed
```
