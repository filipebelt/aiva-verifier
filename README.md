## ğŸ§  About the Project

**AIVA (Artificial Intelligence Verification Assistant)** is an intelligent verification system designed to detect whether images or videos were generated by AI models (such as diffusion models or deepfake techniques) or captured from the real world.

AIVA combines:

- computer vision analysis  
- noise and artifact inspection  
- embedding-based comparison  
- synthetic pattern detection  
- preprocessing pipelines  

to deliver reliable, probability-based authenticity results.
